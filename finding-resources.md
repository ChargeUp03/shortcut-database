# Finding Resources

When submitting your shortcut, you'll need some resources. You must have an iCloud link, but having a .shortcut file, screenshots, and a shortcut icon are not required but reccommended. How to get each of these are detailed below.

## iCloud Links

iCloud links are required to submit a shortcut, but you do not need to include one if you include a .shortcut file. You can include either one or both.

iCloud links can be retrieved very easily, as this is the official way of sharing shortcuts.
1. Open a shortcut with the (...) button
2. Select the share ( ) button
3. Select 'Copy iCloud Link'
4. Select 'Copy Link'

After selecting 'Copy Link', a new dialogue box will appear with a loading icon. Do not select 'Cancel'. Once complete, the loading symbol will turn into a check (✓). The link will be copied to your clipboard. You can then enter it into the submission form.

## .shortcut File

.shortcut files are an alternative way to share shortcuts. You can simply download it and tap on it to add it to your shortcuts. 

.shortcut files require using the Shortcuts app itself to retrieve them. You can use a premade shortcut I made to get these easily.
1. Download the Shortcut File and Icon Exporter shortcut
2. Run the shortcut by tapping anywhere on it besides the (...) icon
3. If you haven't already created a new folder by selecting 'Create Shortcut Exporting Folder'
4. Select '.shortcut File'
5. Follow the instructions in the shortcut
6. Locate your new .shortcut file in Files or Finder

The .shortcut file will be named the same as the shortcut itself and will include the shortcut's icon. You can rename the shortcut, but do not rename the .shortcut extension.

## Shortcut Icons

A shortcut icon is the small icon and background gradient you can set in the Shortcuts app. It is possible to save this icon as a .png file using a shortcut.

As with retrieving a .shortcut file, you will also need to download the shortcut I made to get your icon.
1. Download the Shortcut File and Icon Exporter shortcut
2. Run the shortcut by tapping anywhere on it besides the (...) icon
3. Select Shortcut Icon
4. Follow the instructions in the shortcut
5. [On iPhone, iPad, and Vision Pro] Open Photos; the shortcut icon should be the newest photo there.
5.1. [On Mac] Locate the location you saved the photo in with Finder; it should be named whatever you named the shortcut, followed by '.PNG'

## Screenshots

While not required, it is recommended to add screenshots demonstrating what your shortcut will do. You can upload 0-10 screenshots when submitting. Do not crop or edit these in any way, unless you are blurring out sensitive information or content.

Screenshotting requires pressing multiple buttons/keys at the same time.

### iPhone (Dynamic Island, notch) & iPad (no home button)
1. Press the side or top (power) and volume up buttons at the same time
1.1. If you have Full-Screen Previews on, you may need to select the check (✓) button, then Save to Photos
2. The screenshot will be saved in Photos

### iPhone (home button) & iPad (home button)
1. Press the side or top button (power) and Home Button at the same time
1.1. If you have Full-Screen Previews on, you may need to select the check (✓) button, then Save to Photos
2. The screenshot will be saved in Photos

### CarPlay
*Most shortcut actions will not affect CarPlay, besides Open App and Show Notification, and potentially some others.
1. While connected to CarPlay, take a screenshot on your iPhone. You must have CarPlay Screenshots enabled.
2. Both a screenshot of your iPhone and CarPlay will be saved to Photos.

### Apple Watch
*You must have Enable Screenshots enabled in your Apple Watch settings.
1. Press the Digital Crown and Side Button at the same time
2. The screenshot will be saved to your iPhone's Photos app

### Vision Pro
1. Press the Digital Crown and Top Button at the same time
2. A snapshot of your view when taking the screenshot will be saved to your Vision Pro's Photos app

### Mac
1. Press Command (⌘) + Shift + 3 to capture the whole screen, or Command (⌘) + Shift + 4 to select a part of the screen
1.1. If you used Command (⌘) + Shift + 4, select the area with your cursor
2. The screenshot will be saved to your Mac's desktop.
